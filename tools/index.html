Our favourites

List of great tools that we use and can recommend. Include flexible, modular, open-source tools that are extremely useful in the according situations (software, hardware, and beyond).

Feel free to send suggestions and feedback, do a speech about the according software or hardware product, or just browse around and gain knowledge.

Full transparency and neutrality note: all the tools in this section are our personal recommendations. None of this section or any other sections of this website is sponsored by anyone.  

Hardware

  • Framework Laptops
  • MNT Reform - Dev ARM Laptop
  • Raspberry Pi, Orange Pi (prototyping)
  • Steam Deck (mobility, gaming)

Software

  • Arch Linux (literally used for everything)
  • Firefox web browser
  • Publii (used to make this website)
  • Suckless Toolkit - recommended software that heavily follows KISS principles

Maker software

  • OpenSCAD - parametric modelling
  • Cura - 3D print preparation & slicing
  • Blender - modelling
